# 🔥 小红书爆款 finder

一个 Chrome 浏览器扩展，帮助你在小红书上快速发现**低粉高赞**的爆款文章。

## ✨ 功能特性

- 🔍 **关键词搜索** - 在小红书搜索页面快速查找相关内容
- 📊 **智能分析** - 自动分析文章的点赞数、收藏数、粉丝数
- 🎯 **爆款识别** - 基于点赞/粉丝比、收藏/粉丝比识别爆款文章
- 🎨 **视觉标注** - 用不同颜色高亮标注超爆款和潜力爆款
- 📥 **数据导出** - 支持将爆款数据导出为 CSV 文件

## 🚀 安装方法

### 方法一：手动安装（推荐）

1. 下载或克隆本仓库
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 打开右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目的文件夹

## 📖 使用说明

### 方式一：搜索页标注（推荐）

1. **访问小红书搜索页** - 打开 [www.xiaohongshu.com/search](https://www.xiaohongshu.com/search)
2. **搜索关键词** - 在小红书站内搜索框输入关键词（如：护肤、穿搭、美食）
3. **点击插件图标** - 点击浏览器工具栏的插件图标
4. **点击「开始搜索」** - 插件会标注所有搜索结果笔记
   - ⚪ **灰色虚线框** = 待分析（需要点击查看详情）
5. **点击笔记进入详情页** - 选择感兴趣的笔记查看

### 方式二：详情页分析

1. **打开笔记详情页** - 点击任意笔记进入详情
2. **点击插件图标** - 点击「开始搜索」按钮
3. **查看分析结果** - 右上角会显示分析徽章：
   - 🔴 **超爆款** = 点赞/粉丝比 ≥ 10 且 收藏/粉丝比 ≥ 5
   - 🟠 **潜力爆款** = 满足部分条件
   - 📄 **普通笔记** = 未达到爆款标准

### 调整判定标准

在插件弹窗中可以调整：
- **点赞/粉丝比** - 默认 ≥ 10
- **收藏/粉丝比** - 默认 ≥ 5
- **最少点赞数** - 默认 ≥ 100

## ⚙️ 爆款判定标准

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 点赞/粉丝比 | ≥ 10 | 点赞数除以作者粉丝数的比值 |
| 收藏/粉丝比 | ≥ 5 | 收藏数除以作者粉丝数的比值 |
| 最少点赞数 | ≥ 100 | 文章的最少点赞数门槛 |

### 超爆款判定
- 点赞/粉丝比 ≥ 设定值
- 收藏/粉丝比 ≥ 设定值
- 点赞数 ≥ 最小值

### 潜力爆款判定
- 点赞/粉丝比 ≥ 设定值 × 0.7 **或** 收藏/粉丝比 ≥ 设定值 × 0.7
- 点赞数 ≥ 最小值 × 0.5

## 🛠️ 开发说明

### 技术栈
- **Manifest V3** - 最新的 Chrome 扩展 API
- **Vanilla JavaScript** - 无框架依赖
- **CSS3** - 现代化样式

### 本地调试

1. 按照「安装方法」加载扩展
2. 打开 `chrome://extensions/`，找到插件，点击「检查视图 - Service Worker」打开开发者工具
3. 在小红书页面上右键 → 检查，可以查看 Content Script 的日志

### 修改配置

在 `popup.html` 中调整默认值：
```javascript
likeRatio: 10,      // 点赞/粉丝比阈值
collectRatio: 5,    // 收藏/粉丝比阈值
minLikes: 100       // 最少点赞数
```

## ⚠️ 注意事项

1. **数据准确性** - 粉丝数、点赞数等数据来自页面显示，可能存在延迟
2. **页面结构变化** - 小红书网页版更新可能导致选择器失效，需要更新 `content.js`
3. **使用限制** - 请遵守小红书的使用条款，不要过度频繁请求
4. **隐私安全** - 本扩展不会收集或上传任何用户数据
5. **搜索卡片限制** - 小红书搜索卡片不直接显示互动数据（点赞/收藏/粉丝数），需要点击笔记查看详情才能获取完整数据进行分析
